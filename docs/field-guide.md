## What to Capture

Each screenshot needs:
1. Rock type, Mass, Resistance, Instability
2. Mineral composition (%)
3. Equipment used
4. Green zone size (S/M/L)
5. Red zone size (S/M/L)

**Save to:** `Rock Info Tool\Training Data`

---

## Screenshot Naming

```
<test>_<rock>_<notes>.png
```

Examples:
- `G1_granite_baseline.png`
- `G2_granite_okunis.png`
- `E1_shale_hofstede.png`

## Zone Size Guide

Estimate green/red zones as:
- **S** = Small (~10-20% of bar)
- **M** = Medium (~25-40% of bar)
- **L** = Large (~50%+ of bar)
